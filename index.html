Inventory system

This inventory system was a programming assignment we had to do in our first year in school. Back then I made a scuffed version with crappy coding and designing. I decided to remake this with slightly less scuffed code. I made the system in a way that others could use it. It's easy to change the UI designs and to change the amount of slots needed in the inventory. I made this in 1,5 hours.
